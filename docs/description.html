# 🎵 RGX | Final Fantasy Level-Up! ![FFLU Logo](https://raw.githubusercontent.com/donniedice/FFLU/main/images/FFLU_logo_400x400.png) ## ![](https://raw.githubusercontent.com/donniedice/SimpleQuestPlates/main/images/kiwi.gif) RGX Mods - RealmGX Community Project *** ## 🎯 Overview **RGX | Final Fantasy Level-Up!** transforms your leveling experience with the iconic **"Victory Fanfare!"** from Final Fantasy — bringing that triumphant feeling to World of Warcraft across all versions! ![RealmGX Kiwi](https://raw.githubusercontent.com/donniedice/SimpleQuestPlates/main/images/kiwi.gif) **The Kiwi Says:** "Victory Fanfare! Bwwiiiee." *** ## ✨ Features 🎵 Custom FF Sound - The authentic Final Fantasy Victory Fanfare plays every time you level up! 🎚️ Quality Options - Choose between High, Medium, and Low quality variants. 🔇 Smart Muting - Auto-mutes WoW's default level-up sound so you get only the fanfare! 🌍 Multi-Language - Supports English, German, French, Spanish, and Russian with automatic detection. 💾 Persistent Config - Settings saved automatically between sessions. ⚡ Performance Optimized - Efficient memory usage and modern WoW addon architecture. 🛡️ Error Handling - Robust protection against crashes throughout. 🎮 All WoW Versions - Supports Retail, Mists, Cataclysm, Wrath, Burning Crusade, and Classic Era! **Note:** All quality variants are included — no additional downloads required. *** ## 📋 Slash Commands The addon can be managed using the following slash commands:
/fflu help # Show all available commands
/fflu test # Play test sound
/fflu enable # Enable addon
/fflu disable # Disable addon (unmutes default sound)
/fflu high # Use high quality sound
/fflu med # Use medium quality sound (default)
/fflu low # Use low quality sound
